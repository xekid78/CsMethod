# CsMethod
メソッド（.NET Core）

## 処理
`sumGokei()`メソッドを作り、その中で計算を行う。

## コード
```
using System;

namespace Method
{
    class Method
    {
        static void SumGokei()
        {
            int sum = 5 * 8;
            Console.WriteLine(sum);
        }

        static void Main(string[] args)
        {
            SumGokei();
        }
    }
}
```

## 出力例  
```
40
```
